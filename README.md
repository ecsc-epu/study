# ECSC Learning Roadmap

Đây là nền tảng tài nguyên học tập **miễn phí** được xây dựng dành riêng cho thành viên Câu lạc bộ An toàn thông tin ECSC (EPU Cyber Security Club).

- 🔰 **Beginner:** Nhập môn An Ninh Mạng
- 🛡️ **Blue Team:** Phòng thủ, giám sát và ứng phó
- ⚔️ **Red Team:** Tấn công và đánh giá bảo mật

## Cập nhật nội dung (Dành cho Admin)

1. Thêm/sửa nội dung bài học trong thư mục `data/courses/*.json`.
2. Mở Terminal và chạy lệnh sau để Build lại giao diện mới:
   ```bash
   node convert.js
   ```
3. Mở file `index.html` hoặc sử dụng Live Server để xem kết quả.
